Detailed Description

Класс, отвечающий за взаимодействие клиента с графическим интерфейсом и сервером.

Класс FunctionsForClient предоставляет методы для аутентификации, регистрации и отправки задач с использованием графического интерфейса. Также он обрабатывает сообщения от сервера.
