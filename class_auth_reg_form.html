Detailed Description

Класс формы авторизации и регистрации.

Этот класс представляет собой окно для авторизации и регистрации пользователей. Он наследуется от QWidget и предоставляет функционал для работы с полями ввода, кнопками и сигналами для отправки данных на сервер.
